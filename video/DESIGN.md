# Portfolio Review Promo Design

## Style Prompt

A sober premium-finance motion presentation that moves from a dark portfolio application into a clean analytical report. The mood is precise, trusted, and evidence-led: graphite app surfaces, olive-green performance signal, crisp report cards, navy headers, and restrained motion that feels like an analyst assembling proof.

## Colors

- Graphite background: `#090a08`
- Deep panel: `#14150f`
- Soft ink: `#f1f4ec`
- Muted text: `#8c9288`
- Signal green: `#8aa52a`
- Report white: `#f7f9fb`
- Report navy: `#244f86`
- Report line: `#dbe2ea`
- XTB red cue: `#e30613`
- Warning red: `#e84b3c`

## Logo

The video source uses the repo-owned `XTB Skills` mark from `video/assets/brand/xtb-skills-mark.svg`, derived from the canonical files in `assets/brand/`. The mark is an original X-shaped agent skill network with XTB-red outer nodes, a central execution node, and a restrained green signal line.

## Typography

- Primary: `Inter`, `Arial`, sans-serif
- Data/monospace: `IBM Plex Mono`, `SFMono-Regular`, monospace

## Motion Rules

- Use smooth push-ins, card reveals, line drawing, and highlight sweeps.
- Keep entrances between 0.35s and 0.75s with `power2.out` or `power3.out`.
- Use decisive but calm transitions; no jitter, glitch, robots, neon brains, or generic AI imagery.
- The video must work without sound, with concise on-screen copy at each beat.

## What NOT To Do

- Do not imply market prediction, trading advice, or autonomous investing.
- Do not use purple/cyan AI gradients or sci-fi robot imagery.
- Do not clutter the frame with too many identical cards.
- Do not obscure the report details when using screenshots as proof.
