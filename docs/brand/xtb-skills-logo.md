# XTB Skills Logo

The XTB Skills identity uses an original agent-skill node mark: a compact X-shaped network with a central execution point, XTB-red outer nodes, and a restrained green workflow signal. It is meant to suggest portfolio data, repeatable skill workflows, and agent handoff without copying the XTB broker logo.

## Assets

- `assets/brand/xtb-skills-mark.svg` - square icon and favicon source.
- `assets/brand/xtb-skills-lockup-dark.svg` - primary lockup for dark README, video, and social preview surfaces.
- `assets/brand/xtb-skills-lockup-light.svg` - lockup for light documents and exported pages.

## Usage

- Use the full `XTB Skills` lockup in README, documentation, and video title/outro frames.
- Use the standalone mark where space is tight, such as badges, favicons, avatars, or small cards.
- Keep the mark original and repo-owned. Do not replace it with the XTB broker logo or mimic the broker's exact identity system.
- Use the red outer nodes as the only explicit XTB color cue. The dominant palette remains graphite, soft ink, and signal green.

## Palette

- Graphite: `#090a08`
- Deep panel: `#14150f`
- Soft ink: `#f1f4ec`
- Muted text: `#8c9288`
- Signal green: `#8aa52a`
- Report white: `#f7f9fb`
- Report line: `#dbe2ea`
- XTB red cue: `#e30613`
- Warning/accent red: `#e84b3c`
